**Goal:** Build Lumen, the Aegis display compositor — a single userspace process that owns the framebuffer, manages windows with z-ordering, renders a software cursor, dispatches input, and includes a PTY-based terminal emulator.

**Architecture:** Single dynamically-linked binary. Polling event loop (non-blocking reads on keyboard, mouse, PTY master fds with 16ms nanosleep). Backbuffer composite with save-under cursor on the frontbuffer. Built-in terminal window via PTY fork+exec of oksh.

### Task 1: Kernel — O_NONBLOCK support for /dev/mouse and PTY master

**Files:**
- Modify: `kernel/fs/initrd.c`
- Modify: `kernel/fs/pty.c`
- Modify: `kernel/syscall/sys_io.c` (if needed to pass flags)

The current `mouse_read_fn` always calls `mouse_read_blocking()` which sleeps. The PTY master read also blocks. Lumen needs non-blocking reads on both. The VFS `read` callback signature is `int (*read)(void *priv, void *buf, uint64_t off, uint64_t len)` — it does NOT receive the fd flags. The flags live on `vfs_file_t.flags`. We need to pass a way for the read function to know about O_NONBLOCK.

**Approach:** Since the `read` callback can't see the fd flags, the simplest fix is to check at the `sys_read` level: if `f->flags & O_NONBLOCK` and the device would block, return `-EAGAIN`. But that requires the read function to distinguish "would block" from "blocking".

The cleanest approach: change `mouse_read_fn` to check if there are events available first (via `mouse_poll`), and if not, return `-EAGAIN`. Then `sys_read` does NOT call `mouse_read_blocking` at all — the blocking behavior comes from `sys_read` retrying (which it doesn't do). So for non-blocking: `mouse_read_fn` returns `-EAGAIN` if empty. For blocking: the existing blocking path still works because `sys_read` calls `f->ops->read()` which calls `mouse_read_blocking()`.

Wait — the simplest approach: add a `mouse_read_fn_nonblock` that uses `mouse_poll` and attach it differently. But that's messy.

**Actual simplest approach:** The vfs_file_t has a `flags` field. Pass the whole `vfs_file_t *` to the read function... but that changes the interface. Too invasive.

**Pragmatic approach for v0.1:** Lumen doesn't use `O_NONBLOCK` at the kernel level. Instead, Lumen polls first with a non-blocking API, then only reads when data is available. For mouse: open `/dev/mouse` normally, use `read()` with a small timeout or check a signal. For keyboard: use `tcsetattr` with `VMIN=0 VTIME=0` which makes `read` return 0 immediately if no data. For PTY master: same VMIN/VTIME trick.

Actually, **musl's `read()` with VMIN=0 VTIME=0** on a terminal fd returns 0 if no data is available. This works for the console TTY (keyboard). For the PTY master, same thing applies.

For `/dev/mouse`: the simplest kernel fix is to make `mouse_read_fn` check `mouse_poll()` first and return 0 if empty AND the caller set a flag. But we can't pass flags through the VFS read interface.

**Final decision:** Add a global `mouse_poll` path. Lumen will call a helper that does a non-blocking check. Since we control both kernel and userspace, the pragmatic approach is:

1. Change `mouse_read_fn` to: try `mouse_poll()` first. If events available, return them. If not, return 0 (EOF-like) instead of blocking. This changes the behavior for ALL readers of /dev/mouse — but Lumen is the only one that reads it.
2. Actually, returning 0 means EOF which is wrong. Let's just make mouse_read_fn always non-blocking: drain available events, return count. If zero events available, return -EAGAIN. `sys_read` already propagates negative returns as errno.

Let's go with that.

- [ ] **Step 1: Modify `mouse_read_fn` in `kernel/fs/initrd.c` to be non-blocking**

Replace the current `mouse_read_fn`:

```c
static int
mouse_read_fn(void *priv, void *buf, uint64_t off, uint64_t len)
{
    (void)priv; (void)off;
    uint32_t count = 0;
    uint32_t max_events = (uint32_t)(len / sizeof(mouse_event_t));

    if (max_events == 0) return 0;

    /* Non-blocking: drain available events only */
    mouse_event_t evt;
    while (count < max_events && mouse_poll(&evt)) {
        __builtin_memcpy((uint8_t *)buf + count * sizeof(mouse_event_t),
                         &evt, sizeof(mouse_event_t));
        count++;
    }

    if (count == 0) return -11;  /* -EAGAIN */
    return (int)(count * sizeof(mouse_event_t));
}
```

This makes `/dev/mouse` always non-blocking. `read()` returns events if available, or -EAGAIN if empty. Lumen's polling loop handles this naturally.

- [ ] **Step 2: Make PTY master read non-blocking when VMIN=0**

The PTY master read path in `pty.c` already handles the case where the buffer is empty — it blocks via `sched_block()`. For Lumen, the PTY master fd will have `VMIN=0 VTIME=0` set via `tcsetattr`, and the kernel's TTY layer (tty_read in `kernel/fs/pty.c`) should respect that. Check if this works — if tty_read already checks for `VMIN=0` and returns 0 instead of blocking, no change needed. If not, add the check.

Read `kernel/fs/pty.c` to find the master read function. If it blocks unconditionally when the buffer is empty, add a check: if the tty is in raw mode with `VMIN=0`, return 0 immediately instead of blocking.

**Note:** This step may not need changes if Lumen just handles `read()` returning -EAGAIN or 0. The PTY master write-side (shell output) goes through a ring buffer. If the ring buffer is empty, master read should ideally return 0 or -EAGAIN rather than blocking, since Lumen polls in a loop anyway.

```c
/* draw.c — Framebuffer drawing primitives for Lumen compositor
 *
 * All functions operate on a surface_t (buffer + dimensions).
 * Extracted from fb_test/main.c with surface parameter added.
 */
#include "draw.h"
#include "terminus20.h"
#include <string.h>

void
draw_px(surface_t *s, int x, int y, uint32_t c)
{
    if (x >= 0 && x < s->w && y >= 0 && y < s->h)
        s->buf[y * s->pitch + x] = c;
}

void
draw_fill_rect(surface_t *s, int x, int y, int w, int h, uint32_t c)
{
    int i, j;
    for (j = y; j < y + h && j < s->h; j++)
        for (i = x; i < x + w && i < s->w; i++)
            if (i >= 0 && j >= 0)
                s->buf[j * s->pitch + i] = c;
}

void
draw_rect(surface_t *s, int x, int y, int w, int h, uint32_t c)
{
    int i;
    for (i = x; i < x + w; i++) {
        draw_px(s, i, y, c);
        draw_px(s, i, y + h - 1, c);
    }
    for (i = y; i < y + h; i++) {
        draw_px(s, x, i, c);
        draw_px(s, x + w - 1, i, c);
    }
}

void
draw_gradient_v(surface_t *s, int x, int y, int w, int h,
                uint32_t c1, uint32_t c2)
{
    int i, j;
    for (j = 0; j < h; j++) {
        int t = j * 255 / (h > 1 ? h - 1 : 1);
        uint32_t r = ((c1>>16&0xFF)*(255-t) + (c2>>16&0xFF)*t) / 255;
        uint32_t g = ((c1>>8&0xFF)*(255-t) + (c2>>8&0xFF)*t) / 255;
        uint32_t b = ((c1&0xFF)*(255-t) + (c2&0xFF)*t) / 255;
        uint32_t c = (r<<16)|(g<<8)|b;
        for (i = 0; i < w; i++)
            draw_px(s, x + i, y + j, c);
    }
}

void
draw_char(surface_t *s, int x, int y, char ch, uint32_t fg, uint32_t bg)
{
    const uint8_t *glyph = &font_terminus[(unsigned char)ch * FONT_H * 2];
    int row, col;
    for (row = 0; row < FONT_H; row++) {
        uint16_t bits = ((uint16_t)glyph[row * 2] << 8) | glyph[row * 2 + 1];
        for (col = 0; col < FONT_W; col++) {
            uint32_t c = (bits & (0x8000 >> col)) ? fg : bg;
            draw_px(s, x + col, y + row, c);
        }
    }
}

void
draw_text(surface_t *s, int x, int y, const char *str,
          uint32_t fg, uint32_t bg)
{
    while (*str) {
        draw_char(s, x, y, *str, fg, bg);
        x += FONT_W;
        str++;
    }
}

void
draw_text_t(surface_t *s, int x, int y, const char *str, uint32_t fg)
{
    while (*str) {
        const uint8_t *glyph = &font_terminus[(unsigned char)*str * FONT_H * 2];
        int row, col;
        for (row = 0; row < FONT_H; row++) {
            uint16_t bits = ((uint16_t)glyph[row * 2] << 8) | glyph[row * 2 + 1];
            for (col = 0; col < FONT_W; col++)
                if (bits & (0x8000 >> col))
                    draw_px(s, x + col, y + row, fg);
        }
        x += FONT_W;
        str++;
    }
}

void
draw_blit(surface_t *dst, int dx, int dy,
          const uint32_t *src, int sw, int sh)
{
    int x, y;
    for (y = 0; y < sh; y++) {
        int ty = dy + y;
        if (ty < 0 || ty >= dst->h) continue;
        for (x = 0; x < sw; x++) {
            int tx = dx + x;
            if (tx < 0 || tx >= dst->w) continue;
            dst->buf[ty * dst->pitch + tx] = src[y * sw + x];
        }
    }
}
```

```c
/* cursor.c — Software cursor with save-under
 *
 * The cursor sprite is a 16x16 monochrome arrow.
 * - mask[]: 1 = pixel is part of cursor, 0 = transparent
 * - color[]: 1 = white, 0 = black (only meaningful where mask=1)
 *
 * Save-under: before drawing cursor, save the 16x16 rect from the
 * framebuffer. On move, restore saved pixels, then save+draw at new pos.
 */
#include "cursor.h"
#include <string.h>

/* 16x16 arrow cursor — each row is a uint16_t bitmask */
static const uint16_t cursor_mask[16] = {
    0xC000, /* 11.............. */
    0xF000, /* 1111............ */
    0xFC00, /* 111111.......... */
    0xFF00, /* 11111111........ */
    0xFFC0, /* 1111111111...... */
    0xFFF0, /* 111111111111.... */
    0xFFFC, /* 11111111111111.. */
    0xFFF0, /* 111111111111.... */
    0xFFC0, /* 1111111111...... */
    0xFF00, /* 11111111........ */
    0xFE00, /* 1111111......... */
    0xEF00, /* 1110.1111....... */
    0xC780, /* 11...11110...... */
    0x0780, /* .....11110...... */
    0x03C0, /* ......1111...... */
    0x0180, /* .......11....... */
};

static const uint16_t cursor_color[16] = {
    0x8000, /* 1_.............. */
    0xC000, /* 11.............. */
    0xE000, /* 111............. */
    0xF800, /* 11111........... */
    0xFC00, /* 111111.......... */
    0xFE00, /* 1111111......... */
    0xFF00, /* 11111111........ */
    0xFE00, /* 1111111......... */
    0xFC00, /* 111111.......... */
    0xF600, /* 1111.11......... */
    0xE300, /* 111...11........ */
    0xC180, /* 11.....11....... */
    0x01C0, /* .......111...... */
    0x01C0, /* .......111...... */
    0x00C0, /* ........11...... */
    0x0000, /* ................ */
};

static surface_t *s_fb;
static uint32_t s_save[CURSOR_W * CURSOR_H];
static int s_x = -1, s_y = -1;
static int s_visible = 0;

void
cursor_init(surface_t *fb)
{
    s_fb = fb;
    s_visible = 0;
}

void
cursor_hide(void)
{
    if (!s_visible || !s_fb) return;
    /* Restore saved pixels */
    int row, col;
    for (row = 0; row < CURSOR_H; row++) {
        int fy = s_y + row;
        if (fy < 0 || fy >= s_fb->h) continue;
        for (col = 0; col < CURSOR_W; col++) {
            int fx = s_x + col;
            if (fx < 0 || fx >= s_fb->w) continue;
            s_fb->buf[fy * s_fb->pitch + fx] = s_save[row * CURSOR_W + col];
        }
    }
    s_visible = 0;
}

void
cursor_show(int x, int y)
{
    if (!s_fb) return;
    s_x = x;
    s_y = y;
    /* Save pixels under cursor */
    int row, col;
    for (row = 0; row < CURSOR_H; row++) {
        int fy = y + row;
        for (col = 0; col < CURSOR_W; col++) {
            int fx = x + col;
            if (fx >= 0 && fx < s_fb->w && fy >= 0 && fy < s_fb->h)
                s_save[row * CURSOR_W + col] = s_fb->buf[fy * s_fb->pitch + fx];
            else
                s_save[row * CURSOR_W + col] = 0;
        }
    }
    /* Draw cursor sprite */
    for (row = 0; row < CURSOR_H; row++) {
        uint16_t m = cursor_mask[row];
        uint16_t c = cursor_color[row];
        int fy = y + row;
        if (fy < 0 || fy >= s_fb->h) continue;
        for (col = 0; col < CURSOR_W; col++) {
            if (!(m & (0x8000 >> col))) continue;  /* transparent */
            int fx = x + col;
            if (fx < 0 || fx >= s_fb->w) continue;
            uint32_t color = (c & (0x8000 >> col)) ? 0x00FFFFFF : 0x00000000;
            s_fb->buf[fy * s_fb->pitch + fx] = color;
        }
    }
    s_visible = 1;
}

void
cursor_move(int x, int y)
{
    cursor_hide();
    cursor_show(x, y);
}
```

```c
/* compositor.c — Window management, chrome, and compositing for Lumen
 *
 * Z-order: windows[0] = bottom, windows[nwindows-1] = top.
 * Composite: background → windows (back to front) → taskbar → memcpy to FB.
 * Cursor is drawn on the framebuffer directly (not in backbuffer).
 */
#include "compositor.h"
#include "cursor.h"
#include "draw.h"
#include <string.h>
#include <stdlib.h>

void
comp_init(compositor_t *c, uint32_t *fb, uint32_t *backbuf,
          int w, int h, int pitch)
{
    memset(c, 0, sizeof(*c));
    c->fb.buf = fb;
    c->fb.w = w;
    c->fb.h = h;
    c->fb.pitch = pitch;
    c->back.buf = backbuf;
    c->back.w = w;
    c->back.h = h;
    c->back.pitch = pitch;
    c->cursor_x = w / 2;
    c->cursor_y = h / 2;
    c->needs_redraw = 1;
}

void
comp_add_window(compositor_t *c, window_t *win)
{
    if (c->nwindows >= MAX_WINDOWS) return;
    c->windows[c->nwindows++] = win;
    c->focused = win;
    c->needs_redraw = 1;
}

void
comp_remove_window(compositor_t *c, window_t *win)
{
    int i, found = -1;
    for (i = 0; i < c->nwindows; i++) {
        if (c->windows[i] == win) { found = i; break; }
    }
    if (found < 0) return;
    /* Shift remaining windows down */
    for (i = found; i < c->nwindows - 1; i++)
        c->windows[i] = c->windows[i + 1];
    c->nwindows--;
    if (c->focused == win)
        c->focused = c->nwindows > 0 ? c->windows[c->nwindows - 1] : NULL;
    if (win->pixels) free(win->pixels);
    free(win);
    c->needs_redraw = 1;
}

void
comp_raise_window(compositor_t *c, window_t *win)
{
    int i, found = -1;
    for (i = 0; i < c->nwindows; i++) {
        if (c->windows[i] == win) { found = i; break; }
    }
    if (found < 0 || found == c->nwindows - 1) return;
    for (i = found; i < c->nwindows - 1; i++)
        c->windows[i] = c->windows[i + 1];
    c->windows[c->nwindows - 1] = win;
    c->needs_redraw = 1;
}

window_t *
comp_window_at(compositor_t *c, int x, int y)
{
    int i;
    /* Search from top to bottom */
    for (i = c->nwindows - 1; i >= 0; i--) {
        window_t *w = c->windows[i];
        if (!w->visible) continue;
        int total_h = w->h + TITLEBAR_HEIGHT + 2 * BORDER_WIDTH;
        int total_w = w->w + 2 * BORDER_WIDTH;
        if (x >= w->x && x < w->x + total_w &&
            y >= w->y && y < w->y + total_h)
            return w;
    }
    return NULL;
}

/* Draw window chrome + client area onto a surface */
static void
draw_window(surface_t *s, window_t *w, int focused)
{
    int total_w = w->w + 2 * BORDER_WIDTH;
    int total_h = w->h + TITLEBAR_HEIGHT + 2 * BORDER_WIDTH;

    /* Drop shadow */
    draw_fill_rect(s, w->x + SHADOW_OFFSET, w->y + SHADOW_OFFSET,
                   total_w, total_h, C_SHADOW);

    /* Window body (white) */
    draw_fill_rect(s, w->x, w->y, total_w, total_h, C_WIN);

    /* Border */
    draw_rect(s, w->x, w->y, total_w, total_h, C_BORDER);

    /* Title bar gradient */
    uint32_t t1 = focused ? C_TITLE1 : C_UTITLE1;
    uint32_t t2 = focused ? C_TITLE2 : C_UTITLE2;
    draw_gradient_v(s, w->x + 1, w->y + 1, total_w - 2, TITLEBAR_HEIGHT,
                    t1, t2);

    /* Title text */
    draw_text(s, w->x + 12, w->y + 6, w->title, C_TITLE_T, t1);

    /* Window buttons (traffic light) */
    if (w->closeable) {
        int bx = w->x + total_w - 80;
        int by = w->y + 6;
        draw_fill_rect(s, bx + 52, by, 20, 18, C_RED);
        draw_text(s, bx + 56, by - 1, "x", C_WIN, C_RED);
        draw_fill_rect(s, bx + 28, by, 20, 18, C_GREEN);
        draw_fill_rect(s, bx + 4, by, 20, 18, C_YELLOW);
    }

    /* Client area blit */
    if (w->pixels) {
        draw_blit(s, w->x + BORDER_WIDTH,
                  w->y + TITLEBAR_HEIGHT + BORDER_WIDTH,
                  w->pixels, w->w, w->h);
    }
}

void
comp_composite(compositor_t *c)
{
    int i;

    /* 1. Desktop background gradient */
    draw_gradient_v(&c->back, 0, 0, c->back.w, c->back.h, C_BG1, C_BG2);

    /* 2. Draw windows back-to-front */
    for (i = 0; i < c->nwindows; i++) {
        window_t *w = c->windows[i];
        if (!w->visible) continue;
        /* Call on_draw to let window update its pixels */
        if (w->on_draw) w->on_draw(w);
        draw_window(&c->back, w, w == c->focused);
    }

    /* 3. Taskbar */
    taskbar_draw(&c->back, c->back.w, c->back.h);

    /* 4. Flip backbuffer to framebuffer */
    memcpy(c->fb.buf, c->back.buf,
           (size_t)c->fb.pitch * (size_t)c->fb.h * 4);

    /* 5. Draw cursor on framebuffer (after flip) */
    cursor_show(c->cursor_x, c->cursor_y);
}

/* Check if click is on the close button of a window */
static int
hit_close_button(window_t *w, int x, int y)
{
    if (!w->closeable) return 0;
    int total_w = w->w + 2 * BORDER_WIDTH;
    int bx = w->x + total_w - 80 + 52;
    int by = w->y + 6;
    return (x >= bx && x < bx + 20 && y >= by && y < by + 18);
}

/* Check if click is in the title bar (but not close button) */
static int
hit_titlebar(window_t *w, int x, int y)
{
    int total_w = w->w + 2 * BORDER_WIDTH;
    return (x >= w->x && x < w->x + total_w &&
            y >= w->y && y < w->y + TITLEBAR_HEIGHT);
}

void
comp_handle_mouse(compositor_t *c, uint8_t buttons, int16_t dx, int16_t dy)
{
    /* Update cursor position */
    c->cursor_x += dx;
    c->cursor_y += dy;
    if (c->cursor_x < 0) c->cursor_x = 0;
    if (c->cursor_y < 0) c->cursor_y = 0;
    if (c->cursor_x >= c->fb.w) c->cursor_x = c->fb.w - 1;
    if (c->cursor_y >= c->fb.h) c->cursor_y = c->fb.h - 1;

    int left = buttons & 1;
    int prev_left = c->prev_buttons & 1;

    /* Dragging */
    if (c->dragging) {
        if (left) {
            c->drag_win->x = c->cursor_x - c->drag_dx;
            c->drag_win->y = c->cursor_y - c->drag_dy;
            c->needs_redraw = 1;
        } else {
            c->dragging = 0;
        }
        goto done;
    }

    /* Left button pressed (transition) */
    if (left && !prev_left) {
        window_t *w = comp_window_at(c, c->cursor_x, c->cursor_y);
        if (w) {
            /* Close button? */
            if (hit_close_button(w, c->cursor_x, c->cursor_y)) {
                comp_remove_window(c, w);
                goto done;
            }
            /* Title bar? Start drag */
            if (hit_titlebar(w, c->cursor_x, c->cursor_y)) {
                c->dragging = 1;
                c->drag_win = w;
                c->drag_dx = c->cursor_x - w->x;
                c->drag_dy = c->cursor_y - w->y;
            }
            /* Focus + raise */
            if (w != c->focused) {
                comp_raise_window(c, w);
                c->focused = w;
                c->needs_redraw = 1;
            }
            /* Forward click to client area */
            if (w->on_mouse) {
                int cx = c->cursor_x - w->x - BORDER_WIDTH;
                int cy = c->cursor_y - w->y - TITLEBAR_HEIGHT - BORDER_WIDTH;
                if (cx >= 0 && cy >= 0 && cx < w->w && cy < w->h)
                    w->on_mouse(w, buttons, cx, cy);
            }
        }
    }

done:
    c->prev_buttons = buttons;
    /* Always redraw cursor position (save-under dance happens in composite) */
    if (dx != 0 || dy != 0)
        c->needs_redraw = 1;
}

void
comp_handle_key(compositor_t *c, char key)
{
    if (c->focused && c->focused->on_key)
        c->focused->on_key(c->focused, key);
}
```

```c
/* terminal.c — PTY-based terminal emulator window for Lumen
 *
 * Opens /dev/ptmx → gets master fd + /dev/pts/N slave.
 * Forks a child, sets slave as stdin/stdout/stderr, execs /bin/oksh.
 * Master fd is polled in the main loop; output bytes are rendered
 * into a text grid (cols × rows) on the window's pixel buffer.
 *
 * No ANSI escape parsing in v0.1 — raw text, newline, carriage return,
 * backspace only.
 */
#include "terminal.h"
#include "draw.h"
#include <stdlib.h>
#include <string.h>
#include <unistd.h>
#include <fcntl.h>
#include <stdio.h>
#include <sys/ioctl.h>

typedef struct {
    int master_fd;
    int cols, rows;
    int cx, cy;             /* cursor position in grid */
    char *grid;             /* cols * rows character buffer */
} term_priv_t;

/* Redraw the terminal's pixel buffer from the text grid */
static void
term_render(window_t *win)
{
    term_priv_t *tp = (term_priv_t *)win->priv;
    if (!tp || !win->pixels) return;

    /* Create a surface for the window's pixel buffer */
    surface_t s = { win->pixels, win->w, win->h, win->w };

    /* Clear to background */
    draw_fill_rect(&s, 0, 0, win->w, win->h, C_TERM_BG);

    /* Draw text grid */
    int row, col;
    for (row = 0; row < tp->rows; row++) {
        for (col = 0; col < tp->cols; col++) {
            char ch = tp->grid[row * tp->cols + col];
            if (ch && ch != ' ') {
                draw_char(&s, col * FONT_W, row * FONT_H, ch,
                          C_TERM_FG, C_TERM_BG);
            }
        }
    }

    /* Draw cursor (solid block) */
    if (tp->cx < tp->cols && tp->cy < tp->rows) {
        draw_fill_rect(&s, tp->cx * FONT_W, tp->cy * FONT_H,
                       FONT_W, FONT_H, C_TERM_FG);
        char ch = tp->grid[tp->cy * tp->cols + tp->cx];
        if (ch && ch != ' ')
            draw_char(&s, tp->cx * FONT_W, tp->cy * FONT_H, ch,
                      C_TERM_BG, C_TERM_FG);
    }
}

/* Scroll the text grid up by one row */
static void
term_scroll(term_priv_t *tp)
{
    memmove(tp->grid, tp->grid + tp->cols,
            (size_t)(tp->rows - 1) * (size_t)tp->cols);
    memset(tp->grid + (tp->rows - 1) * tp->cols, ' ', (size_t)tp->cols);
    tp->cy = tp->rows - 1;
}

static void
term_on_key(window_t *self, char key)
{
    term_priv_t *tp = (term_priv_t *)self->priv;
    if (!tp) return;
    /* Write keystroke to PTY master → shell receives it */
    write(tp->master_fd, &key, 1);
}

static void
term_on_draw(window_t *self)
{
    term_render(self);
}

void
terminal_write(window_t *win, const char *data, int len)
{
    term_priv_t *tp = (term_priv_t *)win->priv;
    if (!tp) return;

    int i;
    for (i = 0; i < len; i++) {
        char ch = data[i];

        if (ch == '\n') {
            tp->cy++;
            if (tp->cy >= tp->rows) term_scroll(tp);
            continue;
        }
        if (ch == '\r') {
            tp->cx = 0;
            continue;
        }
        if (ch == '\b' || ch == 127) {
            if (tp->cx > 0) tp->cx--;
            continue;
        }
        if (ch == '\t') {
            tp->cx = (tp->cx + 8) & ~7;
            if (tp->cx >= tp->cols) { tp->cx = 0; tp->cy++; }
            if (tp->cy >= tp->rows) term_scroll(tp);
            continue;
        }

        /* Printable character */
        if (ch >= 32 && ch < 127) {
            tp->grid[tp->cy * tp->cols + tp->cx] = ch;
            tp->cx++;
            if (tp->cx >= tp->cols) {
                tp->cx = 0;
                tp->cy++;
                if (tp->cy >= tp->rows) term_scroll(tp);
            }
        }
    }
}

window_t *
terminal_create(int cols, int rows, int *master_fd_out)
{
    /* Open PTY master */
    int master_fd = open("/dev/ptmx", O_RDWR);
    if (master_fd < 0) return NULL;

    /* Get slave path (grantpt/unlockpt are no-ops on Aegis) */
    int pts_num = 0;
    ioctl(master_fd, 0x80045430 /* TIOCGPTN */, &pts_num);
    char slave_path[32];
    snprintf(slave_path, sizeof(slave_path), "/dev/pts/%d", pts_num);

    /* Allocate window */
    window_t *win = (window_t *)calloc(1, sizeof(window_t));
    if (!win) { close(master_fd); return NULL; }

    win->w = cols * FONT_W;
    win->h = rows * FONT_H;
    win->pixels = (uint32_t *)calloc((size_t)win->w * (size_t)win->h, 4);
    if (!win->pixels) { free(win); close(master_fd); return NULL; }

    snprintf(win->title, sizeof(win->title), "Terminal");
    win->visible = 1;
    win->closeable = 1;
    win->on_key = term_on_key;
    win->on_draw = term_on_draw;

    /* Allocate terminal private data */
    term_priv_t *tp = (term_priv_t *)calloc(1, sizeof(term_priv_t));
    tp->master_fd = master_fd;
    tp->cols = cols;
    tp->rows = rows;
    tp->grid = (char *)calloc((size_t)cols * (size_t)rows, 1);
    memset(tp->grid, ' ', (size_t)cols * (size_t)rows);
    win->priv = tp;

    /* Set master fd non-blocking */
    fcntl(master_fd, F_SETFL, O_NONBLOCK);

    /* Fork child process for shell */
    pid_t pid = fork();
    if (pid == 0) {
        /* Child: open slave as stdin/stdout/stderr */
        close(master_fd);
        setsid();
        int slave_fd = open(slave_path, O_RDWR);
        if (slave_fd < 0) _exit(1);
        dup2(slave_fd, 0);
        dup2(slave_fd, 1);
        dup2(slave_fd, 2);
        if (slave_fd > 2) close(slave_fd);
        /* Set TERM env var */
        /* Exec shell */
        char *argv[] = { "/bin/oksh", NULL };
        char *envp[] = { "TERM=dumb", "HOME=/root", "PATH=/bin", NULL };
        execve("/bin/oksh", argv, envp);
        _exit(1);
    }

    *master_fd_out = master_fd;
    return win;
}
```

```c
/* lumen — Aegis display compositor (Phase 37)
 *
 * Owns the framebuffer, manages windows, dispatches input.
 * v0.1: single process, built-in terminal + system info windows.
 */
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <unistd.h>
#include <fcntl.h>
#include <termios.h>
#include <time.h>
#include <sys/syscall.h>

#include "draw.h"
#include "cursor.h"
#include "compositor.h"
#include "terminal.h"

typedef struct {
    uint64_t addr;
    uint32_t width, height, pitch, bpp;
} fb_info_t;

typedef struct __attribute__((packed)) {
    uint8_t  buttons;
    int16_t  dx;
    int16_t  dy;
    int16_t  scroll;
} mouse_event_t;

static struct termios s_orig_termios;

static void
restore_terminal(void)
{
    tcsetattr(0, TCSANOW, &s_orig_termios);
}

/* Create a system info window (static content, like fb_test) */
static window_t *
create_sysinfo_window(int fb_w, int fb_h)
{
    int cols = 35;
    int rows = 10;
    int cw = cols * FONT_W;
    int ch = rows * FONT_H;

    window_t *win = (window_t *)calloc(1, sizeof(window_t));
    if (!win) return NULL;
    win->w = cw;
    win->h = ch;
    win->pixels = (uint32_t *)calloc((size_t)cw * (size_t)ch, 4);
    if (!win->pixels) { free(win); return NULL; }

    snprintf(win->title, sizeof(win->title), "System Information");
    win->visible = 1;
    win->closeable = 1;

    /* Draw info content onto the window's pixel buffer */
    surface_t s = { win->pixels, cw, ch, cw };
    uint32_t bg = 0x00F0F4F8;
    draw_fill_rect(&s, 0, 0, cw, ch, bg);

    int lh = FONT_H + 4;
    int x = 10, y = 10;
    draw_text(&s, x, y, "SYSTEM", C_ACCENT, bg);
    y += lh + 2;
    draw_fill_rect(&s, x, y, cw - 20, 1, 0x00C0C8D0);
    y += 6;

    draw_text(&s, x, y, "Kernel:   Aegis 0.1 (x86_64)", C_TEXT, bg); y += lh;
    draw_text(&s, x, y, "Arch:     x86_64 (AMD64)", C_TEXT, bg); y += lh;
    {
        char buf[48];
        snprintf(buf, sizeof(buf), "Display:  %dx%d @ 32bpp", fb_w, fb_h);
        draw_text(&s, x, y, buf, C_TEXT, bg);
    }
    y += lh;
    draw_text(&s, x, y, "Security: Capability-based", C_TEXT, bg); y += lh;
    draw_text(&s, x, y, "Shell:    oksh (OpenBSD ksh)", C_TEXT, bg); y += lh;
    draw_text(&s, x, y, "Init:     Vigil supervisor", C_TEXT, bg);

    return win;
}

int
main(void)
{
    fb_info_t fb_info;
    memset(&fb_info, 0, sizeof(fb_info));
    long ret = syscall(513, &fb_info);
    if (ret < 0 || fb_info.addr == 0) {
        fprintf(stderr, "lumen: no framebuffer (ret=%ld)\n", ret);
        return 1;
    }

    int fb_w = (int)fb_info.width;
    int fb_h = (int)fb_info.height;
    int fb_pitch = (int)(fb_info.pitch / 4);
    uint32_t *fb = (uint32_t *)(uintptr_t)fb_info.addr;

    /* Allocate backbuffer */
    size_t fb_bytes = (size_t)fb_pitch * (size_t)fb_h * 4;
    uint32_t *backbuf = (uint32_t *)malloc(fb_bytes);
    if (!backbuf) {
        fprintf(stderr, "lumen: cannot allocate backbuffer\n");
        return 1;
    }

    /* Set stdin to raw mode (no echo, no line buffering, no signals) */
    tcgetattr(0, &s_orig_termios);
    atexit(restore_terminal);
    struct termios raw = s_orig_termios;
    raw.c_lflag &= ~(unsigned)(ECHO | ICANON | ISIG);
    raw.c_cc[VMIN] = 0;
    raw.c_cc[VTIME] = 0;
    tcsetattr(0, TCSANOW, &raw);

    /* Open mouse device */
    int mouse_fd = open("/dev/mouse", O_RDONLY);
    if (mouse_fd < 0)
        fprintf(stderr, "lumen: warning: no /dev/mouse\n");

    /* Initialize compositor */
    compositor_t comp;
    comp_init(&comp, fb, backbuf, fb_w, fb_h, fb_pitch);

    /* Initialize cursor */
    cursor_init(&comp.fb);

    fprintf(stderr, "[LUMEN] started %dx%d\n", fb_w, fb_h);

    /* Create terminal window */
    int term_cols = (fb_w * 3 / 5) / FONT_W;
    int term_rows = ((fb_h * 3 / 5) - TITLEBAR_HEIGHT) / FONT_H;
    if (term_cols < 40) term_cols = 40;
    if (term_rows < 10) term_rows = 10;

    int term_master_fd = -1;
    window_t *term_win = terminal_create(term_cols, term_rows, &term_master_fd);
    if (term_win) {
        /* Center the terminal window */
        int total_w = term_win->w + 2 * BORDER_WIDTH;
        int total_h = term_win->h + TITLEBAR_HEIGHT + 2 * BORDER_WIDTH;
        term_win->x = (fb_w - total_w) / 2;
        term_win->y = (fb_h - TASKBAR_HEIGHT - total_h) / 2;
        comp_add_window(&comp, term_win);
    }

    /* Create system info window */
    window_t *info_win = create_sysinfo_window(fb_w, fb_h);
    if (info_win) {
        info_win->x = 20;
        info_win->y = 20;
        comp_add_window(&comp, info_win);
    }

    /* Focus terminal */
    if (term_win) {
        comp_raise_window(&comp, term_win);
        comp.focused = term_win;
    }

    /* ── Main event loop ── */
    for (;;) {
        int activity = 0;

        /* Poll keyboard (stdin in raw mode, VMIN=0) */
        {
            char key;
            if (read(0, &key, 1) == 1) {
                comp_handle_key(&comp, key);
                activity = 1;
            }
        }

        /* Poll mouse */
        if (mouse_fd >= 0) {
            mouse_event_t mevt;
            while (read(mouse_fd, &mevt, sizeof(mevt)) == (int)sizeof(mevt)) {
                comp_handle_mouse(&comp, mevt.buttons, mevt.dx, mevt.dy);
                activity = 1;
            }
        }

        /* Poll PTY master (shell output) */
        if (term_master_fd >= 0) {
            char buf[512];
            int n = (int)read(term_master_fd, buf, sizeof(buf));
            if (n > 0) {
                terminal_write(term_win, buf, n);
                comp.needs_redraw = 1;
                activity = 1;
            }
        }

        /* Composite if dirty */
        if (comp.needs_redraw) {
            cursor_hide();
            comp_composite(&comp);
            comp.needs_redraw = 0;
        }

        /* Sleep ~16ms if idle */
        if (!activity) {
            struct timespec ts = {0, 16000000};
            nanosleep(&ts, NULL);
        }
    }

    return 0;
}
```

```python
#!/usr/bin/env python3
"""Phase 37 Lumen compositor test.

Boots QEMU q35 with xHCI + USB mouse + USB keyboard.
Logs in, launches lumen, verifies startup message on serial.
Injects mouse movement and keyboard input, checks for responsiveness.
"""
import subprocess, time, sys, os, select, fcntl, re, socket, tempfile

QEMU = "qemu-system-x86_64"
ISO = "build/aegis.iso"
BOOT_TIMEOUT = int(os.environ.get("BOOT_TIMEOUT", "900"))
CMD_TIMEOUT = 30


def build_iso():
    real_uid = os.getuid()
    real_gid = os.getgid()
    def drop_euid():
        os.setegid(real_gid)
        os.seteuid(real_uid)
    r = subprocess.run(["make", "iso"], preexec_fn=drop_euid)
    if r.returncode != 0:
        print("[FAIL] make iso failed")
        sys.exit(1)


def read_until(proc, pattern, deadline):
    fd = proc.stdout.fileno()
    fl = fcntl.fcntl(fd, fcntl.F_GETFL)
    fcntl.fcntl(fd, fcntl.F_SETFL, fl | os.O_NONBLOCK)
    buf = b""
    while time.time() < deadline:
        ready, _, _ = select.select([proc.stdout], [], [], 0.5)
        if not ready:
            continue
        try:
            chunk = os.read(fd, 4096)
        except BlockingIOError:
            continue
        if not chunk:
            break
        buf += chunk
        text = buf.decode("utf-8", errors="replace")
        if re.search(pattern, text):
            return text
    return buf.decode("utf-8", errors="replace")


def send_keys(mon, text):
    for c in text:
        if c == '\n':
            cmd = "sendkey ret"
        elif c == ' ':
            cmd = "sendkey spc"
        elif c == '/':
            cmd = "sendkey slash"
        elif c == '-':
            cmd = "sendkey minus"
        elif c == '_':
            cmd = "sendkey shift-minus"
        elif c == '&':
            cmd = "sendkey shift-7"
        elif c == '.':
            cmd = "sendkey dot"
        else:
            cmd = f"sendkey {c}"
        mon.sendall((cmd + "\n").encode())
        time.sleep(0.08)


def main():
    build_iso()

    mon_path = tempfile.mktemp(suffix=".sock")
    qemu_cmd = [
        QEMU, "-machine", "q35", "-m", "2G",
        "-cdrom", ISO, "-boot", "order=d",
        "-display", "none", "-vga", "std",
        "-nodefaults", "-serial", "stdio", "-no-reboot",
        "-device", "qemu-xhci,id=xhci",
        "-device", "usb-mouse,bus=xhci.0",
        "-device", "usb-kbd,bus=xhci.0",
        "-monitor", f"unix:{mon_path},server,nowait",
        "-device", "isa-debug-exit,iobase=0xf4,iosize=0x04",
    ]

    proc = subprocess.Popen(qemu_cmd, stdout=subprocess.PIPE,
                            stderr=subprocess.DEVNULL)
    passed = 0
    total = 2

    try:
        deadline = time.time() + BOOT_TIMEOUT

        # Wait for login prompt
        output = read_until(proc, r"login:", deadline)
        if "login:" not in output:
            print("[FAIL] No login prompt")
            sys.exit(1)

        time.sleep(1)
        mon = socket.socket(socket.AF_UNIX, socket.SOCK_STREAM)
        mon.connect(mon_path)
        time.sleep(0.5)
        mon.recv(4096)

        # Login
        send_keys(mon, "root\n")
        output = read_until(proc, r"password:", time.time() + CMD_TIMEOUT)
        send_keys(mon, "forevervigilant\n")
        output = read_until(proc, r"[#$] ", time.time() + CMD_TIMEOUT)

        # Test 1: Launch lumen, check startup message
        send_keys(mon, "lumen &\n")
        output = read_until(proc, r"\[LUMEN\] started", time.time() + CMD_TIMEOUT)
        if "[LUMEN] started" in output:
            print("[PASS] Lumen started successfully")
            passed += 1
        else:
            print(f"[FAIL] Lumen did not start: {output[-300:]}")

        # Test 2: Inject mouse movement (verify no crash after 2 seconds)
        time.sleep(1)
        mon.sendall(b"mouse_move 50 30\n")
        time.sleep(0.5)
        mon.sendall(b"mouse_move -20 10\n")
        time.sleep(1)

        # If we get here without QEMU crashing, lumen is responsive
        # Check process is still running
        if proc.poll() is None:
            print("[PASS] Lumen handles mouse input without crashing")
            passed += 1
        else:
            print("[FAIL] QEMU exited unexpectedly")

        mon.close()

    finally:
        proc.terminate()
        proc.wait(timeout=5)
        try:
            os.unlink(mon_path)
        except OSError:
            pass

    print(f"\n{passed}/{total} tests passed")
    sys.exit(0 if passed == total else 1)


if __name__ == "__main__":
    main()
```
