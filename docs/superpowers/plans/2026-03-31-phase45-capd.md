1. Add `sys_cap_grant(target_pid, kind, rights)` in `sys_process.c`:

   - Validate kind is 1‚Äì15

2. Add declaration in `sys_impl.h`

## Task 3: capd daemon (userspace)

**Files:** `user/capd/main.c`, `user/capd/Makefile` (new)

Build the capd daemon:

1. **Policy loader:** Read `/etc/aegis/capd.d/*.policy` files at startup. Parse
   `allow CAP1 CAP2 ...` lines. Store in a static table (max 16 policies, each
   with binary basename and allowed cap bitmask).

2. **Cap name parser:** `cap_name_to_kind()` function (same table as stsh).

3. **Socket server:** `socket(AF_UNIX)` ‚Üí `bind("/run/capd.sock")` ‚Üí `listen(8)` ‚Üí
   accept loop.

4. **Request handler:** Per connection:
   - `getsockopt(SO_PEERCRED)` ‚Üí client PID
   - `read /proc/<pid>/exe` ‚Üí basename
   - Read request line: `GRANT <cap_name>\n`
   - Validate against loaded policy
   - `syscall(363, client_pid, kind, 7)` (full rights)
   - Send `OK\n` or `ERR <reason>\n`
   - Log to stderr: `[CAPD] GRANT <cap> -> pid <N> (<binary>) OK/DENIED`

5. **Makefile:** Static binary (musl-gcc), install to `/bin/capd`.

**Verify:** Compiles clean.

**Time:** ~5 minutes
