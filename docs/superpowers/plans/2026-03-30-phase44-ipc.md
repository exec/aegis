## Task 1: AF_UNIX data structures + socket/close (~5 min)

**New files:** `kernel/net/unix_socket.h`, `kernel/net/unix_socket.c`

- Define `unix_sock_t`, `passed_fd_t`, `unix_name_entry_t` structs
- Static table `s_unix_socks[32]` + `s_unix_names[32]` + spinlock
- `unix_sock_alloc()` ‚Äî find free slot, set in_use, return id
- `unix_sock_free()` ‚Äî decrement refcount, free rings on zero, remove name entry
- `unix_sock_get()` ‚Äî bounds-check + in_use check
- VFS ops struct `g_unix_sock_ops` with read/write/close/dup/stat
- Close handler: wake blocked peer, mark CLOSED, decrement refcount
- Dup handler: increment refcount

**Test:** Compiles clean with `make`.

## Task 2: AF_UNIX in sys_socket + bind + listen (~5 min)

**Modify:** `kernel/syscall/sys_socket.c`

- `sys_socket`: Add `AF_UNIX (1)` branch ‚Äî cap_check `CAP_KIND_IPC`, call `unix_sock_alloc()`, install fd with `g_unix_sock_ops`
- `sys_bind`: Detect AF_UNIX fd (check `ops == &g_unix_sock_ops`), parse `sockaddr_un.sun_path`, add to name table, create ramfs stub if path under `/tmp/` or `/run/`
- `sys_listen`: Works generically (set state to LISTENING, store backlog) ‚Äî add AF_UNIX state transition

**Modify:** `kernel/cap/cap.h` ‚Äî add `CAP_KIND_IPC 15u`

**Test:** `socket(AF_UNIX)` + `bind()` + `listen()` succeed in ipc_test stub.

## Task 3: AF_UNIX connect + accept (~5 min)

**Modify:** `kernel/syscall/sys_socket.c`

- `sys_connect`: AF_UNIX path ‚Äî look up path in name table, find listening socket, allocate server-side unix_sock_t, cross-link peer_ids, allocate 2 kva pages for ring buffers, capture peer credentials (pid/uid/gid from sched_current()), enqueue server-side id into listener's accept queue, wake listener
- `sys_accept`: AF_UNIX path ‚Äî block until accept_queue non-empty, dequeue server-side sock_id, create fd for it, fill sockaddr_un if provided

**Test:** Server+client connect succeeds. Minimal round-trip.

## Task 4: AF_UNIX read/write (ring buffer) (~5 min)

**Modify:** `kernel/net/unix_socket.c`

- `unix_sock_read()` ‚Äî read from peer's tx_ring, block if empty (wake writer), return -EPIPE if peer closed
- `unix_sock_write()` ‚Äî write to own tx_ring, block if full (wake reader), return -EPIPE if peer closed
- Ring buffer macros: `RING_USED`, `RING_FREE`, wrapping head/tail

**Test:** Echo test ‚Äî client sends "hello", server reads it, sends it back, client verifies.

## Task 5: SO_PEERCRED (~3 min)

**Modify:** `kernel/syscall/sys_socket.c`

- `sys_getsockopt`: Add `SO_PEERCRED` case for AF_UNIX fds ‚Äî copy `struct ucred { pid, uid, gid }` from unix_sock_t to userspace

**Test:** Server accepts, getsockopt SO_PEERCRED, verify pid > 0.

## Task 6: memfd_create + ftruncate (~5 min)

**New files:** `kernel/fs/memfd.h`, `kernel/fs/memfd.c`

- `memfd_t` struct with phys_pages array, refcount, size
- `memfd_alloc()` ‚Äî find free slot
- VFS ops: read (copy from phys pages via window map), write (copy to phys), close (free pages on refcount zero), dup (refcount++), stat (S_IFREG, size)
- `sys_memfd_create` (syscall 319) ‚Äî cap_check CAP_KIND_IPC, alloc memfd, return fd
- `sys_ftruncate` (syscall 77) ‚Äî validate fd is memfd, allocate/free phys pages to match size

**Modify:** `kernel/syscall/syscall.c` ‚Äî add cases 319 and 77

**Test:** memfd_create + ftruncate + write + read round-trip.

## Task 7: MAP_SHARED (~5 min)

**Modify:** `kernel/syscall/sys_memory.c`

- Remove `MAP_SHARED` rejection (line 202-203)
- Add shared path: if `MAP_SHARED` + fd is memfd, map memfd's physical pages into process page table
- New `VMA_SHARED = 8` in `kernel/mm/vma.h`
- `sys_munmap`: skip `pmm_free_page` for `VMA_SHARED` entries (memfd owns pages)

**Test:** mmap MAP_SHARED on memfd, write through mapping, read back via VFS read, verify match.

## Task 8: sendmsg/recvmsg + SCM_RIGHTS (~5 min)

**Modify:** `kernel/syscall/sys_socket.c`

- `sys_sendmsg`: parse msghdr from user, copy iov data via unix_sock_write, parse cmsghdr for SCM_RIGHTS, dup sender's fds, stage in peer's passed_fds[]
- `sys_recvmsg`: read data via unix_sock_read, if passed_fds pending install into receiver's fd_table, build cmsghdr response, copy_to_user

**Test:** Sender creates memfd, writes magic, passes fd via sendmsg, receiver recvmsg + reads magic.
